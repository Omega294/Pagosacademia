# Academia de Pagos - Gestión en la Nube

Sistema profesional para la gestión de pagos de academia con clases los lunes y martes. Diseñado para ser accesible desde cualquier dispositivo mediante **GitHub Pages** y **Supabase**.

## 🚀 Despliegue Rápido

1.  **Configurar Base de Datos**:
    - Crea un proyecto en [Supabase](https://supabase.com/).
    - Ejecuta el script SQL que se encuentra en el plan de migración (ubicado en los artefactos de este chat).
2.  **Activar GitHub Pages**:
    - Sube este código a tu repositorio `Omega294/Pagosacademia`.
    - Ve a **Settings > Pages** en GitHub.
    - Selecciona la rama `main` y la carpeta `/ (root)`.
    - ¡Listo! Tu app estará en `https://omega294.github.io/Pagosacademia/`.
3.  **Conectar la App**:
    - Una vez abierta la app en tu navegador, haz clic en el icono de **Configuración de Nube** (engranaje) en la parte inferior izquierda.
    - Ingresa tu **Supabase URL** y **Anon Key**.
    - La app se sincronizará automáticamente.

## 📋 Funcionalidades
- **Control de Estudiantes**: Registro y edición de alumnos por categorías.
- **Pagos Multimoneda**: Registro en USD y BS con tasa de cambio configurable.
- **Servicios**: Mensualidades, Membresías, Clases Generales, Amistosos y Material POP.
- **Venta de Mercancía**: Sección dedicada para uniformes y accesorios.
- **Sincronización**: Los datos se guardan en la nube pero funcionan localmente para mayor velocidad.
